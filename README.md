# Dental White — Dental Clinic Landing Page

A modern, conversion-focused landing page built for a real dental clinic in Montevideo, Uruguay.

## 🛠️ Built With
- HTML5
- CSS3

## ✨ Features
- Responsive design for mobile and desktop
- Online appointment request form
- Services section with treatments overview
- Patient testimonials
- Team presentation
- WhatsApp and direct contact integration

## 📁 Project Context
Developed as a freelance project for a dental clinic with 33K+ Instagram followers.
Focused on lead generation and online appointment booking.
